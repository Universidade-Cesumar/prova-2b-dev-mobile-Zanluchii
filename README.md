# Controle de Estoque - Enfermagem

## Descrição

Aplicativo mobile desenvolvido em React Native para auxiliar no controle de estoque de materiais utilizados em atividades de enfermagem.

O sistema permite cadastrar novos materiais e consultar os itens armazenados por meio da integração com a MockAPI.

## Tecnologias Utilizadas

* React Native
* JavaScript
* MockAPI
* Hooks (useState e useEffect)
* FlatList

## Funcionalidades

* Cadastro de materiais
* Consulta de materiais cadastrados
* Atualização automática da lista
* Integração com API externa
* Interface mobile responsiva

## Estrutura do Projeto

* App.js: Tela principal da aplicação.
* MockAPI: Armazenamento e consulta dos dados.
* README.md: Documentação do projeto.

## Como Executar

1. Instalar as dependências:

npm install

2. Iniciar o projeto:

npm start

3. Executar no emulador ou dispositivo móvel.
